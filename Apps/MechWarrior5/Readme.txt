1. Copy RzInterhaptics to C:\Program Files (x86)\Interhaptics\ (contains Clipboard Haptic Engine and custom haptic effects)
2. Copy the contents of MechWarrior5 to C:\Program Files\Epic Games\MW5Mercs\MW5Mercs\Mods. If the Mods folder does not exist, you must create it. A complete guide with pictures can be found inside the MechWarrior5 folder.
3. Use the Drivers folder to update the necessary drivers and firmware.
